# 構文解析

```
[100 PLUS 200 MINUS 300 EOF]
```
というトークン列から
```
        -
    +       300
100     200
```
というツリー構造を作る

### どうやる

まず、このトークン列は式であり、今回の式は以下の生成規則からなる
```式 = 数字 (記号 数字)*```
よって、これに合わないトークン列はエラーとして処理する  
次に生成するツリー構造について  
例から、演算子なら左の子に振り、その演算子に対応する数値は右の子に振っていく

1. ツリー構造のノードを型定義する
2. トークン列を読み込む
3. トークンを一つ読み、ノードを生成する
4. 生成したノードを適切に分けていく
5. EOFノードまで続ける
